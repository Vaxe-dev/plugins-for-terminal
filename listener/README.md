# Listener Plugin
Speech to text translation.
## Description
```sh
listener [LANG]
```
LANG *(Option)* - Choice of spoken language. (Default: browser language)
## Version
#### 1.0.1
## Download
```
tpm install terminal.orielhaim.tk/plugin/listener
```
